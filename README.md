# 🛡️ Fake Image Detection App

AI-powered Fake Image Detection Application developed using Deep Learning and Computer Vision techniques to identify manipulated or tampered images.

---

## 🚀 Features

- Detects fake/manipulated images
- Deep Learning based prediction
- Confidence score output
- Image upload support
- Real-time image analysis
- Beginner-friendly AI project

---

## 🛠️ Technologies Used

- Python
- Flask
- TensorFlow / CNN
- OpenCV
- Flutter
- Deep Learning

---

## 🎯 Project Objective

The objective of this project is to detect fake or tampered images using Artificial Intelligence and image processing techniques for digital media verification and cybersecurity applications.
